обо мне

Меня зовут Новикова Ольга Витальевна. Живу в Севастополе, родилась в 1992 году. В настоящий момент обучаюсь на 4 курсе в Севастопольском национальном техническом университете на факультете Автоматики и вычислительной техники, кафедра Кибернетики и вычислительной техники. Планирую получить диплом магистра. Уровень владения английским - Intermediate.

Очень хочу стать более грамотным разработчиком веб-интерфейсов; научиться правильно верстать и использовать javascript, тестировать, оптимизировать; научиться проектровать интерфейсы осознанно, а не на интуитивном уровне («наверно, пользователю будет так удобнее»); перенять опыт у специалистов, пообщаться с интересными людьми.

Узнала о ШРИ2013 из Яндекс.Субботник. Готова уделять стажировке максимум времени, работе – 16 - 20 часов в неделю.

знания

Языки программирования: CSS, HTML и HTML5(+APIs), JS, C#, PHP, SQL (версии MySQL и MS SQL).

Паттерны: MVC, MVVM.

Фреймворки: jQuery, Knockout, Backbone, Ext JS, Bootstrap, LESS, Modernizr.

Технологии: AJAX, HTML5 APIs.

Не считаю, что какой-то язык лучше, а какой-то хуже. Всё зависит от поставленной задачи. Но клиентской частью приложений мне нравится заниматься больше, чем серверной. Из фреймворков больше всего нравятся jQuery, Knockout, LESS.

Редакторы и среды разработки: Notepad++, Dreamviewer, Visual Studio 2010, Sublime Text 2(+расширения). В работе пользуюсь Visual Studio 2010, а Sublime Text 2 – дома, для поправок кода.

ОС: Windows - разработка, Linux - пользователь.

Системы контроля версий: SVN – на текущей работе, Git – для проекта на ШРИ 2013. Больше понравился Git, потому что он быстрее, есть возможность создания веток, децентрализация.

Системы постановки и ведения задач: Jira.
